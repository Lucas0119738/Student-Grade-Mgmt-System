# Student-Grade-Mgmt-System
1. Add Student: Insert unique student records (ID, Name, Score) into a `std::map`. Prevents duplicate IDs automatically.
2. List All Students: Displays all records cleanly aligned, automatically sorted by Student ID.
3. Sort by Score: Copies data to a std::vector and sorts students from highest to lowest score using a Lambda expression.
4. Search by ID: Performs an efficient search to locate a student's information.
5. Show Statistics: Calculates class average, identifies highest/lowest scores (powered by custom Template Functions), and counts pass/fail ratios.
